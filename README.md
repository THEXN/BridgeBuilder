# 铺桥插件
一个 TShock 的快速铺桥插件。

## 权限
| 权限名               | 命令                   |
|----------------------|------------------------|
| bridgebuilder.bridge | bridge 或 桥来          |

> 首先，玩家需要手持平台、种植箱或墙团队块。 <br>
> 当玩家使用命令“/bridge”时，服务器将开始在玩家的方向放置平台/种植箱/团队块。 <br>
> 当路径上有方块时，放置过程将停止。 <br>
> 现在可以自定义可用于铺桥的物块id了，注意是物块id
